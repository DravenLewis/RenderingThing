Scene

Declared in src/Scene.h.

Header: src/Scene.h

Base scene type that owns models and lighting, participates in state switching, and renders through View screens.

Usage

Typical usage pattern (adjust to your pipeline):

#include "Scene.h"

class MyScene : public Scene3D {
public:
    using Scene3D::Scene;
    void init() override;
    void update(float dt) override;
    void render() override;
};
